# Small microphone echotest Linux (Seml)

I have created a very small Linux script for testing the microphone output. 

You are welcome to use it. 


## Requirements 

You need the following linux package: 

- pavucontrol


Installation: 

    sudo apt install pavucontrol


## Using 

Put the test on:

    ./testMic.sh on 

Put the test off:

    ./testMic.sh off
